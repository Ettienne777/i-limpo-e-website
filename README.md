# i-lipo | Professional Skincare Slimming & Nails

A modern, animated website for i-lipo Endermologie Aesthetic Clinic built with React, TailwindCSS, and Framer Motion.

## Features

- **Beautiful Hero Section** - Eye-catching design with animations
- **Interactive Services** - Categorized services with pricing displayed in elegant cards
- **Amazing Gallery** - Hover effects and smooth transitions
- **Contact Section** - Complete contact information with WhatsApp integration
- **Floating WhatsApp Button** - Easy client communication
- **Responsive Design** - Works perfectly on all devices
- **Smooth Animations** - Powered by Framer Motion for an alive feel

## Tech Stack

- **React 19** - Modern React with latest features
- **Vite** - Lightning-fast build tool
- **TailwindCSS** - Utility-first CSS framework
- **Framer Motion** - Production-ready motion library for React
- **Lucide React** - Beautiful & consistent icon toolkit

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

### Deploy to GitHub Pages

1. Build the project:
```bash
npm run build
```

2. Push your code to GitHub

3. Go to your repository Settings → Pages

4. Select "Deploy from a branch" and choose `gh-pages` branch (or main branch with `/dist` folder)

### Deploy to Netlify

1. Build the project:
```bash
npm run build
```

2. Drag and drop the `dist` folder to Netlify

Or connect your GitHub repository to Netlify for automatic deployments.

### Deploy to Vercel

1. Push your code to GitHub

2. Import your repository in Vercel

3. Vercel will automatically detect Vite and deploy

## Customization

### Update Services & Prices

Edit the `services` array in `src/App.jsx` to update services, prices, and durations.

### Update Contact Information

Edit the contact details in the Contact section of `src/App.jsx`:
- Phone numbers
- Email addresses
- Address
- Business hours
- WhatsApp number

### Update Brand Colors

Edit the color palette in `tailwind.config.js` under `theme.extend.colors`.

### Add Your Own Images

1. Place your images in the `public/` folder
2. Update the `galleryImages` array in `src/App.jsx` with your image paths
